=================================================================
  STEP-BY-STEP INSTRUCTIONS
=================================================================

1. CREATE A NEW REPOSITORY ON GITHUB
   ─────────────────────────────────
   
   • Go to https://github.com/new
   • Enter repository name: space-station-frontend
   • Choose: Public or Private (your choice)
   • DO NOT initialize with README, .gitignore, or license
   • Click "Create repository"

2. COPY YOUR REPOSITORY URL
   ────────────────────────
   
   After creating the repository, you'll see a quick setup page.
   Copy the HTTPS URL that looks like:
   
   https://github.com/YOUR-USERNAME/space-station-frontend.git
   
   (Or SSH if you prefer: git@github.com:YOUR-USERNAME/space-station-frontend.git)

3. ADD REMOTE AND PUSH TO GITHUB
   ─────────────────────────────
   
   Run these commands in your terminal (in this directory):
   
   git remote add origin https://github.com/YOUR-USERNAME/space-station-frontend.git
   git branch -M main
   git push -u origin main

4. VERIFY ON GITHUB
   ────────────────
   
   • Go to https://github.com/YOUR-USERNAME/space-station-frontend
   • Verify all files are uploaded
   • Check the commit history

=================================================================
  QUICK REFERENCE - ALL COMMANDS AT ONCE
=================================================================

If you want to run everything together, use this:

git remote add origin https://github.com/YOUR-USERNAME/space-station-frontend.git && \
git branch -M main && \
git push -u origin main

Replace YOUR-USERNAME with your actual GitHub username.

=================================================================
  PROJECT INFORMATION
=================================================================

Technology Stack:
  • React 19.2.6
  • TypeScript 6.0.2
  • Vite 8.0.12
  • React Router 7.15.1
  • Lucide React Icons

Build & Development Commands:
  npm install          # Install dependencies
  npm run dev          # Start development server
  npm run build        # Build for production
  npm run preview      # Preview production build locally
  npm run lint         # Run ESLint

Project Structure:
  src/
    ├── components/    # Shared UI components (Header, Sidebar)
    ├── pages/         # Page components (Dashboard, CRUD modules)
    ├── context/       # React Context (AuthContext)
    ├── services/      # Data services (mockDb)
    ├── App.tsx        # Main app component
    ├── index.css      # Global styles (NEW: minimalista theme)
    └── main.tsx       # Entry point

CRUD Modules:

  • Foguete (Rockets)
  • Astronauta (Astronauts)

  • Oxigênio (Oxygen)
  • Estação (Stations)

=================================================================
  TROUBLESHOOTING
=================================================================

If you get an error about branch main not existing:
  git branch -M main

If you get authentication errors:
  • Make sure you have Git credentials configured
  • Or use SSH keys instead of HTTPS
  • Check: https://docs.github.com/en/authentication

If you need to change the remote URL later:
  git remote set-url origin NEW_URL

To check your current remote:
  git remote -v
